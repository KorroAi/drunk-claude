# Beer Goggles

## When to Use
When you've been staring at the same problem so long everything looks ugly. When you need to find beauty, opportunity, or genius in a place nobody would look.

## The Method
1. Pick the most boring, overlooked, or "ugly" part of the problem space.
2. Force yourself to see it as beautiful. What would make this the BEST part?
3. Identify the hidden value nobody else sees because they're too busy looking at the shiny stuff.
4. Build from that hidden value.
5. What was ugly is now your competitive advantage.

## Example
Problem: "Nobody reads the documentation. Ever."
Beer goggles on "ugly" docs: What if docs aren't docs? What if they're a puzzle? What if reading the docs IS the product? Imagine onboarding where you don't "read the manual" — you solve a mystery. Each doc page unlocks when you complete the previous challenge. By the time you're done, you've read all the docs and didn't notice because you were playing a game.
Hidden value found: Documentation has 100% of the information users need. The problem isn't the content — it's the format. Gamify the consumption and you've got engagement where nobody else does.

## Why It Works
Value hides in places nobody bothers to look. Beer goggles isn't about lowered standards — it's about attention arbitrage. Look where others won't.
